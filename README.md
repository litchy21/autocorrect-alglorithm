# autocorrect-alglorithm

Simple autocorrect algorithm in PHP

You can uncomment the last lines to test the script :

#### Test the wordExist function (return if the word exists in the dictionary)
```
var_dump(wordExist('sun'));
```

#### Test the getNearestLetters function (return nearest letters in AZERTY keyboard)
```
var_dump(getNearestLetters('q'));
```

#### Test the autocorrect function (return nearest word(s) found)
```
var_dump(autocorrect('hrllp'));
```
